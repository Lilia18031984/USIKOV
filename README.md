# Expense Tracker - Трекер расходов

**Автор:** Никита Усиков

## 📝 Описание

Expense Tracker - это графическое приложение для отслеживания личных расходов. Позволяет добавлять расходы, просматривать их по категориям и датам, подсчитывать сумму расходов за различные периоды времени. Все данные автоматически сохраняются в JSON-файл.

## ✨ Основные возможности

- 💰 Добавление расходов с указанием суммы, категории, даты и описания
- 📊 Подсчёт суммы расходов за выбранный период (сегодня, неделя, месяц, произвольный)
- 🔍 Фильтрация по категориям и датам
- 📈 Быстрая статистика (расходы сегодня/неделя/месяц)
- ✏️ Редактирование и удаление записей
- 💾 Автоматическое сохранение в JSON
- ✅ Валидация вводимых данных
- 📋 10 предустановленных категорий расходов
