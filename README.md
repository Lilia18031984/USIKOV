# 🐙 GitHub User Finder

Приложение для поиска пользователей GitHub с графическим интерфейсом на Tkinter.

## 👤 Автор
**Усиков Никита** (замените на свои данные)

## 📋 Описание

GitHub User Finder — это десктопное приложение, которое позволяет:
- Искать пользователей GitHub по имени пользователя
- Просматривать подробную информацию о пользователе
- Добавлять пользователей в избранное
- Просматривать список избранных пользователей
- Сохранять избранных между запусками программы
