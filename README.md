# Movie Library - Личная кинотека

**Автор:** Никита Усиков

## 📝 Описание

Movie Library - это графическое приложение для ведения личной коллекции фильмов. Позволяет добавлять, просматривать, редактировать и удалять фильмы, а также фильтровать их по жанру и году выпуска. Все данные сохраняются в JSON-файл.

## ✨ Основные возможности

- ➕ Добавление фильмов (название, жанр, год, рейтинг)
- 📋 Просмотр в удобной таблице
- 🔍 Фильтрация по жанру
- 📅 Фильтрация по диапазону годов
- ✏️ Редактирование информации
- 🗑️ Удаление фильмов
- 📊 Статистика (количество, средний рейтинг, лучший рейтинг)
- 💾 Сохранение в JSON
